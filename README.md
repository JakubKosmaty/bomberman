# Bomberman Game

## Rules

- Similar to original bomberman game - https://en.wikipedia.org/wiki/Bomberman

### Player 1

- Movement - WASD 
- Place Bomb - SPACE

### Player 2

- Movement - ARROWS
- Place Bomb - ENTER
